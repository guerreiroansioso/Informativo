# Informativo

Aplicacao Streamlit com dados ficticios de internacoes e mortes por queimaduras,
classificados por estados do Brasil em grafico de barras e mapa interativo.

## Como executar

1. Crie e ative um ambiente virtual:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Instale as dependencias:

```powershell
pip install -r requirements.txt
```

3. Rode a aplicacao:

```powershell
streamlit run app.py
```
